# NOODS Landing Page

A playful, retro CPG-style landing page for **NOODS** — prebiotic instant noodles.

## Quick Start

Open `index.html` in a browser, or serve locally:

```bash
python3 -m http.server 8080
```

Then visit [http://localhost:8080](http://localhost:8080).
